# Car Rental System

This is a Spring Boot-based Car Rental System project. It provides RESTful APIs for managing cars and rentals.

## Features

- CRUD operations for cars and rentals
- Validation for input data
- Exception handling
- Unit tests for service layer

## Technologies Used

- Java 17
- Spring Boot 3.x
- Spring Data JPA
- H2 Database
- JUnit 5
- Mockito
